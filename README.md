## 后端启动
1. 导入 IDEA，等待 Maven 下载依赖
2. 创建 MySQL 数据库 team_db，执行 schema.sql
3. 修改 application.yml 中的数据库密码
4. 运行 TeamApplication.java
5. 访问 http://localhost:8080/api/team/recommend 测试

## 前端启动
1. 下载微信开发者工具
2. 导入 miniapp 文件夹
3. 在 project.config.json 中填写自己的 AppID
4. 勾选“不校验合法域名”（开发阶段）

# 校园组队匹配小程序

## 项目简介
面向软件学院学生的组队工具，区分「竞赛」与「课业」需求，基于技能标签智能推荐。

## 技术栈
- 后端：Spring Boot 2.7 + MyBatis-Plus + MySQL + JWT
- 前端：微信小程序原生框架
